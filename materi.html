Reconnaissance Tools & Techniques

Active Reconnaissance

Active reconnaissance involves direct interaction with the target system to gather information. Popular tools include:

  • Nmap - Network Mapper
  • Metasploit Framework
  • Wireshark
  • Nuclei - Vulnerability Scanner
  • Gobuster - Directory/File Enumeration

Passive Reconnaissance

Passive reconnaissance involves gathering information without direct interaction with the target. Tools include:

  • Shodan
  • Censys
  • Google Dorks
  • FOFA - Cyber Security Search Engine
  • ZoomEye

WHOIS Tools

WHOIS lookup tools help gather domain registration and ownership information:

  • DomainTools WHOIS
  • WHOIS.com

Domain Information Groper (dig)

DNS lookup utility for querying DNS servers:

  • Google Dig Tool
  • Dig Web Interface

Subdomain Enumeration

Tools for discovering subdomains of a target domain:

  • Sublist3r
  • Subfinder
  • OWASP Amass
  • Assetfinder

DNS Enumeration

Tools for gathering DNS information and performing zone transfers:

  • DNSenum
  • Fierce

Virtual Host Discovery

Tools for discovering virtual hosts on a target server:

  • Virtual Host Discovery
  • FFUF

Fingerprinting

Tools for identifying technologies and services running on target systems:

  • Wappalyzer
  • Netcraft
  • BuiltWith

Certificate Transparency Logs

Tools for discovering subdomains and monitoring SSL/TLS certificates:

  • crt.sh - Certificate Search
  • Google CT Log
  • Facebook CT Monitor

WAF Detection

Tools for identifying and analyzing Web Application Firewalls:

  • Wafw00f - WAF Detection Tool
  • WhatWaf

Vulnerability Scanners

Tools for automated security assessment and vulnerability detection:

  • Nikto - Web Server Scanner
  • Acunetix
  • Nessus
  • Nikto3
  • OWASP ZAP
  • Burp Suite

Advanced Google Dorking

Advanced search techniques for information gathering:

  • Google Hacking Database (GHDB)
  • Pentest-Tools Google Hacking

Web Archives

Tools for accessing historical versions of websites:

  • Internet Archive - Wayback Machine
  • Archive.today
  • Google Cache Viewer

Automated Reconnaissance

All-in-one tools for automated information gathering:

  • FinalRecon - The Last Recon Tool You'll Need
  • ReconDog
  • theHarvester
  • httpx - HTTP Toolkit
  • getallurls (gau)
  • hakrawler - Web Crawler
