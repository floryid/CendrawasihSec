Google Dork Cheat Sheet

Apa itu Google Dork?

Google Dork adalah teknik pencarian khusus menggunakan operator pencarian Google untuk menemukan informasi spesifik yang mungkin tidak mudah ditemukan melalui pencarian biasa. Teknik ini sering digunakan dalam penetration testing dan information gathering.

Peringatan: Gunakan teknik ini dengan bijak dan etis. Jangan menggunakan untuk aktivitas ilegal atau merugikan pihak lain.

Operator Dasar

intext:

Mencari halaman yang mengandung teks tertentu di dalam kontennya.

intext:"password" filetype:txt

Mencari file txt yang mengandung kata "password"

site:

Membatasi pencarian pada domain atau situs tertentu.

site:example.com intext:admin

Mencari halaman yang mengandung kata "admin" di domain example.com

Operator Lanjutan

filetype:

Mencari file dengan tipe tertentu.

filetype:pdf site:gov.id confidential

Mencari file PDF yang mengandung kata "confidential" di situs pemerintah Indonesia

intitle:

Mencari halaman dengan judul tertentu.

intitle:"index of" intext:config

Mencari direktori yang mengandung file konfigurasi

Dork untuk Keamanan

Exposed Database Files

Mencari file database yang terekspos.

ext:sql | ext:dbf | ext:mdb intext:password

Mencari file database yang mungkin mengandung informasi password

Login Pages

Mencari halaman login yang mungkin rentan.

inurl:login site:target.com

Mencari halaman login di domain target.com

Kombinasi Dork

Contoh Kombinasi Efektif

Beberapa contoh kombinasi dork yang sering digunakan:

Tujuan Dork Penjelasan
Mencari File Sensitif site:target.com filetype:pdf intext:(confidential | secret | private) Mencari dokumen PDF yang mengandung kata-kata sensitif
Mencari Panel Admin site:target.com inurl:(admin | administrator | webadmin | cpanel | dashboard) Mencari berbagai jenis halaman admin
File Konfigurasi site:target.com ext:(xml | conf | cnf | reg | inf | rdp | cfg | txt | ora | ini) Mencari berbagai jenis file konfigurasi
